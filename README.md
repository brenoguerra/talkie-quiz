# Quiz Talkie

Um quiz interativo para avaliar necessidades de comunicação e oferecer soluções personalizadas de tradução.

## 🚀 Tecnologias

- Next.js 14 (App Router)
- React 18
- TypeScript 5
- Tailwind CSS 3
- Framer Motion 10
- ESLint + Prettier
- Husky (Git Hooks)

## ✨ Funcionalidades

- Quiz interativo e dinâmico
- Análise de respostas em tempo real
- Interface responsiva e moderna
- Animações fluidas com Framer Motion
- Recomendações personalizadas
- Integração com APIs externas
- Sistema de autenticação
- Dashboard administrativo

## 🛠️ Pré-requisitos

- Node.js 18+ 
- npm ou yarn
- Git

## 🚦 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/quiz-talkie.git
cd quiz-talkie
```

2. Instale as dependências:
```bash
npm install
# ou
yarn install
```

3. Configure as variáveis de ambiente:
```bash
cp .env.example .env.local
```

4. Execute o projeto:
```bash
npm run dev
# ou
yarn dev
```

5. Acesse `http://localhost:3000`

## 📝 Scripts disponíveis

- `npm run dev` - Inicia o ambiente de desenvolvimento
- `npm run build` - Gera build de produção
- `npm run start` - Inicia o servidor de produção
- `npm run lint` - Executa verificação de linting
- `npm run test` - Executa os testes
- `npm run format` - Formata o código com Prettier
